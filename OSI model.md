#Theory 
The OSI model is made up of 7 layers.

### Application
Retrieves data from application, is known as the window for application services to access the network. 

Protocols at this level include SMTP, FTP and DNS.

### Presentation
Encodes data into correct format.

### Session
Establishes/ terminates a session

### Transport
Sends and receives ports as headers, confirms transport of data, which is broken down into segents for delivery to Network layer.

Protocols include TCP and UDP
### Network
Sends segments across network, known as routing, determines a suitable route, keeps IP addresses in the header.


### Data link layer
node-node delivery, transmits it to host using MAC address, frames data, packets are framed and sent to next device, 

hardware includes switches, bridges.

### Physical
Sent as electronic transmissions signals, contvertedinto bits
