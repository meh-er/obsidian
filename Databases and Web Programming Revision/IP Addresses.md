#DAWP 

- IPv4 Address:
	- 32 bit number - no new blocks left
	- Uniquely addresses a **device, many devices (NAT** or a **router**
	- Divided into four octets e.g. 192.168.1.1. or 74.225.135.239
	- Local (100.x.x.x, 172.16.x.x, 192.168.x.x) vs public addresses
- Replaced by IPv6: 128-bit number, so 2^128 or 3.4x10^38 addresses
	- Eight groups fo four hexadecimal digits
