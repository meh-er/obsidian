#DAWP 

Spring Boot is an open-source Java framework used for programming standalone, production-grade Spring-based applications with a bundle of libraries that make project startup and management easier. Spring Boot is a convention-over-configuration extension for the Spring Java platform intended to help minimise configuration concerns while creating Spring-based applications.

- Backend Framework: for Java-based server-side scalable dynamic web apps.
- Dynamic Data: Integrates with databases to deliver real-time, personal content via **RESTful APIs**.
- **Rapid Development as it Provides**:
	- Dependency Injection
	- Embedded servers
	- Auto-configuration,
	- Pre-built components for faster development.

#### Java @Annotations
- Associate **metadata** with **code elements**:
	- Classes, methods, fields, etc.
- Not statements, so do not directly get executed
- Used at compile or runtime by framteworks
- Some annotations you might recognise from year 1:
	- `@Override`: Indicates that a method overrides a method in its super
