### ifconfig
- Network Interface (if) confirguration (config)
- Manage IP addresses, route of current device
- ip: linux/modern; ifconfig: unix/legacy

#### nslookup
- Query DNS Servers - get domain names and IP addresses

#### ping
- Cheks a website, server, device is online and reachable
- Uses ICMP Echo Requests/Replies

#### traceroute
- Shows the route (Sequence of routers) of packets from the machine to a target web server
- Visualises the network *hops* across the internet


#### telnet
- Insecure Protocol for Remote Login
- Released in 1969, replaced by SSH
- Test TCP connections to any service & port
- Send raw commands

#### SSH
- Secure remote terminal to control a server
- Runs on port 22 (Secure alternative to telnet)
- SSH key pairs - access server without manual password entry
