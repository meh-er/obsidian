#Theory 

Dynamic analysis is also known as black-box testing. It involves the software application and observing its behaviour in **real-time**. This technique simulates the actual runtime environment of the software and can uncover issues related to performance, memory leaks, and security vulnerabilities that may only manifest during execution. 
Dynamic analysis tools typically include profilers, debuggers, and fuzzers that help testers identify and diagnose problems in the software.
- Executes the software application to observe its behaviour.
- Simulates the actual runtime environment
- Uncover issues related to performance, memory leaks, and security vulnerabilities
- Includes profilers, debuggers, and fuzzers.
